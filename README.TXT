--------------------------------------------------------------------------
Esse é o jogo Scape Fire, onde o usuário deverá se movimentar utilizando 
as setas do teclado para escapar dos fogos e dosurubus, buscando sempre 
coletar os cogumelos para aumentar a sua pontuação.
--------------------------------------------------------------------------
